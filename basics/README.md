# Basics

This section covers the fundamental concepts of Go programming.

## Topics Covered

### 01. Hello World
- Basic program structure
- Package and import statements
- The main function

### 02. Variables and Constants
- Variable declaration (var, :=)
- Constants
- Type inference
- Zero values

### 03. Data Types
- Basic types (int, float, string, bool)
- Type conversion
- Type aliases

### 04. Operators
- Arithmetic operators
- Comparison operators
- Logical operators
- Bitwise operators

### 05. Input/Output
- Reading user input
- Formatted output
- String formatting

## Learning Approach

Each topic includes:
1. A README explaining the concepts
2. Commented code examples
3. Practical exercises

Start with `01-hello-world` and progress sequentially.
